# 🛍️ Moda Urbana - Website E-commerce

Website de e-commerce moderno e responsivo para loja de roupas e calçados, desenvolvido com React, Vite e Tailwind CSS.

![Moda Urbana](https://images.unsplash.com/photo-1441984904996-e0b6ba687e04?w=1200&h=400&fit=crop)

## ✨ Características

- 🎨 Design moderno e responsivo
- 📱 Totalmente adaptável para mobile, tablet e desktop
- 🛒 Catálogo de produtos com filtros por categoria
- 💬 Integração direta com WhatsApp para vendas
- 📸 Botão de seguir no Instagram
- 🗺️ Informações de localização e contato
- ⚡ Performance otimizada com Vite
- 🎯 Interface intuitiva e fácil navegação

## 🚀 Tecnologias

- [React](https://react.dev) - Biblioteca JavaScript para interfaces
- [Vite](https://vitejs.dev) - Build tool e dev server
- [Tailwind CSS](https://tailwindcss.com) - Framework CSS
- [Lucide Icons](https://lucide.dev) - Ícones SVG
- [shadcn/ui](https://ui.shadcn.com) - Componentes de UI

## 📦 Instalação

```bash
# Clonar o repositório
git clone https://github.com/SEU_USUARIO/loja-moda-urbana.git

# Entrar na pasta
cd loja-moda-urbana

# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev
```
